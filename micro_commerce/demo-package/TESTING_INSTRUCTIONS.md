   - 📁 File: `app-release.apk`
   - 📏 Size: 60.9 MB
   - 🎯 Version: v1.0 Release

2. **Installation Steps:**
   ```
   1. Download APK to your Android device
   2. Open File Manager → Downloads
   3. Tap on "app-release.apk"
   4. Allow "Install from unknown sources" if prompted
   5. Tap "Install"

   ```
